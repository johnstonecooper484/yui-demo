## What is Yui?
Yui is an AI co-partner being built from the ground up — 
a full companion designed for gamers and streamers with 
real-time game tips, Twitch/YouTube chat monitoring, 
persistent memory, voice interaction, internet research, 
and a live coding workroom.

## This Repository
This is the UI mockup only — an interactive demo of the 
planned interface. The backend AI system is under active 
development and is not included here.
